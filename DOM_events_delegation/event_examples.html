JavaScript DOM Events Examples

1. Basic Event Handling


button.addEventListener('click', function() {
    console.log('Button was clicked!');
});

2. Event Object Properties


button.addEventListener('click', function(event) {
    console.log('Event object:', event);
    // Access properties like event.type, event.target, etc.
});

3. Event Bubbling

Click the child button and watch the event bubble up to the parent

Parent Element 

child.addEventListener('click', function(event) {
    console.log('Child clicked!');
    // Uncomment to stop bubbling:
    // event.stopPropagation();
});

parent.addEventListener('click', function(event) {
    console.log('Parent clicked!');
});

4. Event Capturing

The parent captures the event before it reaches the child

Parent Element 

parent.addEventListener('click', function(event) {
    console.log('Parent captured!');
}, true); // true enables capturing phase

child.addEventListener('click', function(event) {
    console.log('Child clicked!');
});

6. Prevent Default Behavior

This link doesn't navigate:

Try to click this link 

link.addEventListener('click', function(event) {
    event.preventDefault();
    console.log('Default link behavior prevented');
});

7. Custom Events

Target element for custom event 

// Listen for custom event
element.addEventListener('awesome', function(event) {
    console.log('Custom event data:', event.detail);
});

// Create and dispatch custom event
const customEvent = new CustomEvent('awesome', {
    bubbles: true,
    detail: { message: 'This is a custom event!' }
});
element.dispatchEvent(customEvent);
